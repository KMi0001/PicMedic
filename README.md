# 📸 PicMedic

AI-powered desktop application for restoring and organizing photos.

## ✨ Features

- 📷 Photo Scan
- 🖼️ HEIC → JPEG Conversion
- 🤖 AI Photo Enhancement
- 🗂️ Metadata Viewer
- 🔍 Duplicate Detection
- 📦 Batch Processing

## 🛠️ Tech Stack

- Electron
- React
- TypeScript
- Python
- OpenCV
